## Description

DOC-069 (commit `0452bb7`, merged in PR #40 on 2026-04-27) added a new H2 `## Custom navigation links` to `docs/Features.md` between the "Recommended panel on the main page" cluster and `## Metadata stale`. Bundling all five into one tiny commit because they share the file and the layout-fix theme, and the cost-of-error is low.

The 2026-04-27 `/review` of DOC-069 marked Gate 7 (Layout) PASS without checking the in-page TOC. The same review session updated `CLAUDE.md` "Documentation Authoring Rules" + Quality Bar Gate 7, the `/implement` skill (Phase 2 step 3 layout bullet), and the `/review` skill (Gate 7) to require in-page TOC sync going forward, and logged this item as the small follow-up that closes the existing drift.
