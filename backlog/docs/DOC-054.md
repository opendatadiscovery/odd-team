## Description

`docs/Adapters.md` is a 32-line orphan page (not referenced by `SUMMARY.md`, so invisible in the sidebar but reachable by direct URL) that duplicates three of the four sections in the new `main-concepts.md`:

- "Adapters" — covered by main-concepts.md "Adapter vs Collector vs Push-client"
- "Push-client" — same
- "Push and pull strategies" — covered by main-concepts.md "Push vs Pull strategies"
