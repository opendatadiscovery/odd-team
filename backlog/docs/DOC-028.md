## Description

Features.md lines 144-145 show the SLA URL as `https://{platform_url}/api/datasets/{dataset_id}/sla`. The OpenAPI spec uses `{data_entity_id}`. While the IDs are functionally the same, the path template mismatch causes confusion for anyone constructing the URL from docs or generating a client.

## Acceptance Criteria

- [ ] Surrounding text (if any) uses the same term consistently
