## Description

The platform supports operator-configurable additional navigation links (`LinksController.java` → `GET /api/links`, backed by `AdditionalLinkProperties`). Administrators can add links (title + URL) that surface in the platform UI — useful for pointing users at internal wikis, runbooks, or dashboards. Zero documentation exists.
