## Description

The platform has an in-app Integration management system for setting up data source connections, entirely undocumented. Surface:

- UI: Management → Integrations, with detail pages for each integration
- API: `IntegrationController.java` — `/api/integrations`, `/api/integrations/{integration_id}`
- Backend: `IntegrationService`

Operators using the UI to configure integrations don't know the feature exists unless they click through Management.
