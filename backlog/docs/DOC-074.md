## Description

`docs/developer-guides/github-organization-overview.md` lists **`odd-airflow`** with a link to `https://github.com/opendatadiscovery/odd-airflow-adapter` (deprecated repo, archived path) alongside the canonical `odd-airflow-2`. The deprecated entry should be retired or rewritten as a "legacy" pointer that explicitly steers readers at `odd-airflow-2`.

Both repo references currently appear back-to-back in the page (`odd-airflow` block immediately followed by `odd-airflow-2` block) — the reader has to compare the descriptions to figure out which one is current.

## Acceptance Criteria

- [ ] The `odd-airflow` block on `docs/developer-guides/github-organization-overview.md` is either:
  - Removed entirely (if the repo is genuinely unused / unmaintained), or
  - Rewritten as a one-sentence "legacy: superseded by `odd-airflow-2`" pointer
- [ ] The `odd-airflow-2` block remains, optionally augmented with the version requirement (Airflow 2.5.1+) for parity with `integrations/push-adapters/odd-airflow-2.md`
- [ ] Cross-link the canonical Airflow integration page (`integrations/push-adapters/odd-airflow-2.md`) from the `odd-airflow-2` block
- [ ] Verify the `odd-airflow-adapter` repo's actual archival status (or current default branch) before deciding remove-vs-rewrite — if it's still alive and accepting issues, "legacy" framing is more accurate than "removed"

## Context

- Discovered during DOC-042 implementation on 2026-04-26 — `navigation/architecture.md` lists only `odd-airflow-2` as the canonical Airflow integration; `odd-airflow` is not in the canonical repo table.
- The page predates the canonical-vocabulary rewrite (DOC-063, 2026-04-25) and the integrations hub (DOC-042). It also predates `odd-airflow-2` reaching maturity.
- Other entries on the page may carry similar drift (e.g., `oddrn-models-package` is described as auto-generating Pydantic models — check that's still true; `odd-collectors` description doesn't mention all four sub-collectors). A wider audit would be a separate item — this one is scoped to the Airflow-specific drift.
