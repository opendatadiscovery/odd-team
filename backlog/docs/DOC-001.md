## Description

A full Azure AD OAuth2 provider is implemented in `odd-platform-api/src/main/resources/application.yml` (lines 128-156) but is completely absent from the OAuth2/OIDC documentation. The docs cover Cognito, GitHub, Google, OKTA, and Keycloak ŌĆö Azure AD is one of the most widely used enterprise identity providers and its omission is a critical coverage gap.

The implementation supports both multi-tenant and single-tenant deployments and exposes 15+ config keys including `azure-tenant-id`, `authorization-uri`, `token-uri`, `user-info-uri`, `admin-attribute`, `admin-principals`, and `admin-groups`.
