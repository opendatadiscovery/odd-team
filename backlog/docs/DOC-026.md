## Description

Features.md lines 27-28 say search supports filtering by "datasources, owners and tags" (3 filters). The actual UI has 7 facet filters: Datasource, Type, Namespace, Owner, Tag, Groups, Statuses. Namespace and Groups filters are particularly load-bearing in multi-team setups and are entirely absent from the docs.
