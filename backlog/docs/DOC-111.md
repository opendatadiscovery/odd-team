### Most impactful

- **Defect Q** (line 179): the joined-command form will FAIL when an operator copy-pastes it. The operator gets `helm install: command not found` or the export silently fails.
- **Defect L** (line 131): the leading-space-inside-quotes form will set the password to a string starting with space — silently misconfigured.
- **Defect N** (line 149): the broken Swagger UI link will 404 on the live site.
