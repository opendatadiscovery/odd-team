## Description

`docs/configuration-and-deployment/odd-platform.md` lines 544-567 contain a "Machine-to-Machine (M2M) Tokens Configuration" section with full YAML and environment-variable config blocks for `auth.s2s.enabled` / `auth.s2s.token`. This is the **third** live copy of the S2S feature — DOC-052 collapsed the `Features.md` copy to a teaser and cross-linked the canonical page (`configuration-and-deployment/enable-security/authentication/s2s.md`), but left this one untouched.

Confirmed live on `https://docs.opendatadiscovery.org/configuration-and-deployment/odd-platform` — renders in full with zero references to the canonical page.

This is a critical-severity fix because the duplicate carries a full config example that contradicts the canonical page's fuller guidance (security considerations, token rotation, HTTPS requirement). A reader landing here misses all of that.
