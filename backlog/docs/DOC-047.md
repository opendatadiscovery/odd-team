## Description

The Directory is a top-level UI page providing a browsable hierarchy (data source types → data sources → entity types → entities) and is completely undocumented. It's a primary navigation mechanism complementary to Search.

Surface:
- UI: top-level `/directory` route, `Directory/` component directory
- API: `DirectoryController.java` — 4 endpoints (`/api/directory`, `/api/directory/datasources`, `/api/directory/datasources/{id}/types`, `/api/directory/datasources/{id}`)
- Backend: `DirectoryService`
