## Description

The platform supports disabling (halting/snoozing) alert notifications per data entity, **per alert type**, for configurable durations (30 minutes to 1 week). Full UI (`AlertTypeRange.tsx`) and API (`GET/PUT /api/dataentities/{id}/alert_config`) exist. The existing mention on `main:Features.md` around line 143 is a single sentence — operators drowning in alerts for one flaky entity can find the mention but not the mechanism.
