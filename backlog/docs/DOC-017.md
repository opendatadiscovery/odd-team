## Description

`build-and-run-odd-collectors.md` line 27 states "Python 3.9.1" as the requirement. This pinning looks like an exact version rather than a minimum, and may be inaccurate. The actual minimum is declared in the monorepo's `pyproject.toml` and should be stated as a range.

## Acceptance Criteria

- [ ] The actual minimum Python version is read from `odd-collectors/pyproject.toml` (and per-sub-collector `pyproject.toml` files if they diverge)
- [ ] `build-and-run-odd-collectors.md` line 27 is updated to reflect that value as a minimum (e.g., "Python 3.9+" or whatever the project actually requires)
- [ ] If the sub-collectors differ in their Python requirements, the docs reflect that difference

## Context

- Finding: F-065 in `findings/docs-accuracy-config-options/2026-04-21-collectors.md`
- Evidence: docs say "Python 3.9.1"; Poetry typically uses version ranges rather than pins
- Severity in scanner: medium

## Implementation Notes

Verify by reading `../odd-collectors/pyproject.toml` directly. If the requirement has changed since "3.9.1" (which is plausible — that version is from 2020), the bump itself is useful news.
