## Description

Query Examples is a first-class feature with top-level UI navigation under "Data Modelling", partially documented (brief Features.md section on `main`) but lacking a dedicated page, full API/RBAC/workflow coverage. Surface:

- UI: `DataModelling/QueryExamples.tsx`, `QueryExampleDetails/`, `QueryExampleForm/`, `QueryExampleSearchResults/`

- Backend: `QueryExampleService`, `QueryExampleSearchService`
- Linking: query examples attach to datasets and to terms
- RBAC: 7 `QUERY_EXAMPLE_*` permissions
