## Description

This is the **third** live copy of the S2S feature — DOC-052 collapsed the `Features.md` copy to a teaser and cross-linked the canonical page (`configuration-and-deployment/enable-security/authentication/s2s.md`), but left this one untouched.

Confirmed live on `https://docs.opendatadiscovery.org/configuration-and-deployment/odd-platform` — renders in full with zero references to the canonical page.

This is a critical-severity fix because the duplicate carries a full config example that contradicts the canonical page's fuller guidance (security considerations, token rotation, HTTPS requirement). A reader landing here misses all of that.
