## Description

`odd-platform.md` line 287 lists `NOTIFICATIONS_RECEIVERS_SLACK_PLATFORM-BASE-URL` as an environment variable. This variable doesn't exist — no code binds it. The correct env var for the platform base URL is `ODD_PLATFORM_BASE_URL` (bound to `odd.platform-base-url`), which is already listed separately. Users copying the Slack-specific one set a variable that is silently ignored.

## Acceptance Criteria

- [ ] The `NOTIFICATIONS_RECEIVERS_SLACK_PLATFORM-BASE-URL` line is removed from the env var section
- [ ] The `ODD_PLATFORM_BASE_URL` entry remains and is referenced from the Slack setup instructions
- [ ] No YAML example still shows the nonexistent key
